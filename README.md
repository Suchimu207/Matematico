# Matemático!

Uma calculadora simples com música, desenvolvida em Python com Pygame e Colorama.

- **Operações matemáticas:**
  - Adição
  - Subtração
  - Multiplicação
  - Exponenciação
  - Divisão
  - Divisão inteira

## Pré-requisitos

- Python 3.8 ou superior

## Instalação

### 1. Clone o repositório
```bash
git clone https://github.com/Suchimu207/Matematico.git
```
### 2. Crie um ambiente virtual
```bash
# Windows
python -m venv .venv
.venv\Scripts\activate

# Linux/Mac
python3 -m venv .venv
source .venv/bin/activate
```
### 3. Instale as dependências
```bash
pip install -r requirements.txt
```
### 4. Execute o programa
```bash
python main.py
```
